# Bike Spare Parts Website

A full-stack **Bike Spare Parts website** built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js) and **JavaScript**. This project provides a seamless platform for managing and purchasing bike spare parts, showcasing modern design and CRUD functionality.

## Features

- **User Authentication**: Login and registration with secure password handling.  
- **Dynamic Product Listings**: View, add, edit, and delete spare parts directly from the database.  
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.  
- **Image Integration**: Bike spare parts images fetched and stored in MongoDB.  
- **Interactive UI/UX**: React-based frontend with real-time updates.  

## Technologies Used

- **Frontend**: React.js, HTML, CSS, JavaScript  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB (Atlas for remote hosting)  
- **State Management**: React hooks and context API

## Setup Instructions

1. Clone the repository:  
   ```bash
   git clone https://github.com/venkatesh14081//bike-spareparts-website.git
   cd bike-spareparts-website
   ```
2. Install dependencies for both frontend and backend:  
   ```bash
   npm install
   cd client && npm install
   ```
3. Start the development server:  
   ```bash
   npm run dev
   ```
